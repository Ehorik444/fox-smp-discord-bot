# 🎮 Fox SMP Discord Bot

Полнофункциональный Discord бот для приёма заявок на Minecraft SMP сервер Fox SMP через красивые модальные окна.

## ✨ Возможности

- 📋 **Модальное окно с 5 полями**
  - Ник в Minecraft
  - Возраст (с валидацией 8-100)
  - Откуда узнали о сервере
  - Ник друга для розыгрышей
  - О себе (длинное описание)

- 🎨 **Красивое оформление**
  - Цветные эмбеды
  - Профессиональный дизайн
  - Быстрые ответы

- 🛡️ **Безопасность**
  - Полная валидация данных
  - Проверка возраста
  - Обработка ошибок
  - Защита от спама

- 📊 **Функциональность**
  - Автоматическая отправка в канал модераторов
  - Логирование событий
  - Настраиваемые команды

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Получение токена Discord бота

1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Нажмите "New Application"
3. Назовите приложение (например, "Fox SMP Bot")
4. Перейдите на вкладку "Bot"
5. Нажмите "Add Bot"
6. Скопируйте токен (нажмите "Copy")

### 3. Получение ID сервера и канала

```
GUILD_ID - ID вашего Discord сервера:
- Включите режим разработчика (User Settings → Advanced → Developer Mode)
- Щелкните правой кнопкой на сервере → Copy Server ID

APPLICATION_CHANNEL_ID - ID канала для заявок:
- Щелкните правой кнопкой на канале → Copy Channel ID
```

### 4. Конфигурация

```bash
# Скопируйте файл примера
cp .env.example .env

# Отредактируйте .env и вставьте ваши данные
```

### 5. Запуск бота

```bash
python main.py
```

## 📝 Использование команд

### На Discord сервере:

```
!setup      - Установить красивую панель заявок
!apply      - Отправить кнопку для подачи заявки
!ping       - Проверить задержку бота
```

## ❌ Решение проблем

### Бот не запускается
- Проверьте, что установлены все зависимости: `pip install -r requirements.txt`
- Убедитесь, что токен правильно вставлен в `.env`
- Проверьте, что создан файл `.env` (не `.env.example`)

### Кнопка заявки не работает
- Убедитесь, что канал `APPLICATION_CHANNEL_ID` существует и бот имеет доступ
- Проверьте права бота на сервере (должны быть "Administrator" или минимум необходимые права)
- Перезагрузите бота: остановите и заново запустите

### Данные не отправляются в канал
- Проверьте, что `APPLICATION_CHANNEL_ID` правильный
- Убедитесь, что бот может писать в этот канал
- Проверьте консоль бота на наличие ошибок

### Возраст не валидируется
- Убедитесь, что возраст вводится числом (не буквами)
- Возраст должен быть от 8 до 100 лет

## 🔐 Безопасность

**ВАЖНО!** Никогда не коммитьте `.env` файл с реальными токенами!

Файл `.gitignore` автоматически исключает:
- `.env` - Ваш файл с конфиденциальными данными
- `__pycache__/` - Кэш Python
- `venv/` - Виртуальное окружение
- `.idea/` - IDE конфигурация
